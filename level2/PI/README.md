### 题目：PI（2014级陈飞宇大神挖坑）

### 要求：

1. 计算PI，精度>1万位（够黑啊！）
1. 打印总时间
1. 打印内存消耗的最大值

### 参考：

[2014级柱神作品](https://github.com/holdzhu/uestc2015ccpp/tree/master/Pi) （做完本题后再看啊！！！）

本目录提供 Python 版本 `calc_pi.py` 与纯 C 版本 `calc_pi.c`（基于 spigot 算法）。

编译 C 程序：

```bash
gcc calc_pi.c -o calc_pi
```

运行：

```bash
./calc_pi
```
